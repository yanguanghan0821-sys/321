## 输出流程

1. 从中国竞彩网抓取最新SP数据（采集时间精确到分钟）
2. 重新打开sporttery.cn二次核对SP是否有变化（2026-05-06用户纠正）
3. 加载联赛深度知识库，获取每支球队当前形势
4. 生成分析报告（按场次001→009顺序）
5. 同步发微信（MEDIA: 标签） + 聊天内Markdown表格

| 要求 | 内容 |
|:-----|:------|

## 工作表结构

建议3个工作表：

### Sheet1: 竞彩推荐总表（主体）

按6个板块顺序排列：

1. **头部信息**: ⏰ 时间标题 + 📡 数据来源 + ⏱ 停售提示 + 🏷 场次总览
2. **联赛形势一览表**: 每场的对决双方、当前排名、联赛形势、战意评估
3. **核心SP数据表**: 每场竞彩SP（主/平/客）、让球、让球SP、返还率
4. **推荐表**: **按场次编号顺序（001→009）排列**，不是按推荐优先级！
   列：序号→场次→方向→信心→首选比分→备选比分→首选半全场→仓位→理由
5. **串关表**: 必须包含三类：
   - ⚽ 胜平负串关（从稳到博的阶梯方案 A稳胆→B稳健→C博冷→D高赔→E全餐）
   - 🥅 比分串关（高赔玩法 A入门2串1→B二线2串1→C高倍2串1）
   - 🔄 **半全场串关（用户2026-05-06明确要求必须加！）**
     - D 半全场A：两场单选（如胜-胜×胜-胜），1注2元，SP~5-6倍
     - E 半全场B⭐：一场单选+一场双选防平（如胜-胜×平-负），2注4元，SP~10倍
6. **核心信号表**: 关键信号+场次+解读+应对策略

### Sheet2: 复式方案A-E

三类玩法分别列：

| 类别 | 方案数量 | 说明 |
|:-----|:--------:|:------|
| ⚽ 胜平负复式 | A-E (5档) | 稳胆→稳健→博冷→高赔→全餐 |
| 🥅 比分复式 | A-C (3档) | 入门→二线→高倍 |
| 🔄 半全场复式 | **A-C (3档) 必加！** | 每场标注半全场选择，单选/双选组合 |

半全场典型选择逻辑：
- 主场强势队 → 胜-胜(首选) / 平-胜(防平)
- 客队优势明显 → 平-负(首选) / 负-负(防黑)
- 平局预期高 → 平-平(首选) / 胜-平(防先赢后平)

### Sheet3: 任选9场方案（如适用）

3胆+双选/全包方案拆解，72元方案常见

## 样式规范

| 元素 | 样式 |
|:-----|:------|
| 表头 | 深蓝 #1A3C6E + 白色粗体 |
| 数据行 | 白色/F5F7FA交替 |
| 高信心推荐(7/10以上) | 绿色底色 E8F5E9 |
| 深盘/保级信号 | 黄色底色 FFF8E1 |
| 不推荐/极度粘连 | 红色底色 FFEBEE |
| ⭐首选方案 | 绿色底色 |
| 边框 | 浅灰 D0D0D0 细线 |
| 列宽 | 根据内容自适应 |

## 推荐呈现规则

1. **按场次编号顺序排列**（001→009），不按推荐优先级排序
2. 每场必须包含：方向+信心指数(X/10)+首选比分+备选比分+首选半全场+仓位
3. 串关必须包含三类：胜平负串关 + 比分串关 + **半全场串关**
4. 数据来源标注精确到分钟
5. 生成后先核对二次抓取数据（用户2026-05-06要求\"检查一下数据对不对再发\"）
