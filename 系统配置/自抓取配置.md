# 自抓取系统说明

## 系统架构

```
定时任务（每天自动执行）
      ↓
Hermes Agent 自动运行
      ↓
┌─────────────────────────────────────┐
│ 1. 打开 500.com                     │
│ 2. 点击"竞彩足球" → 提取数据        │
│ 3. 点击"足彩胜负" → 提取14场数据    │
│ 4. 调用 Python 脚本写入 Excel       │
│ 5. 调用 Python 脚本写入竞彩分析库   │
│ 6. 生成报告并推送                   │
└─────────────────────────────────────┘
```

## 使用的脚本（推荐）

| 脚本 | 路径 | 功能 |
|:----|:-----|:-----|
| `daily_sync.py` | `~/.hermes/scripts/daily_sync.py` | **一站式写入**：同时更新总表Excel + 竞彩分析库 |
| `obsidian_sync.py` | `~/.hermes/scripts/obsidian_sync.py` | 写入竞彩分析库（Obsidian格式） |

## 每日同步命令（推荐方式 — 推荐优先使用）

分析完比赛后，**一行命令**同时完成所有写入：

```bash
/usr/bin/python3 ~/.hermes/scripts/daily_sync.py all 2026-05-06 '<比赛数据JSON>' '<分析数据JSON>'
```

这个命令会自动：
1. 把比赛数据追加到总表「竞彩数据」（类型=竞彩）
2. 把主力推荐写入总表（类型=推荐）
3. 把14场数据写入总表（类型=14场）
4. 把分析报告写入竞彩分析库的「每日预测」

> 注意：总表只有**一个工作表**「竞彩数据」，所有类型的数据通过「类型」列区分。不再使用多工作表结构。

### 比赛数据JSON格式
```json
[{"league":"英超","home":"利物浦","away":"阿森纳","win_odds":"1.72","draw_odds":"3.50","lose_odds":"4.00",
  "hdcp":"半球","h_water":"0.95","a_water":"0.90","time":"18:00","match_no":"周一001"}]
```

### 分析数据JSON格式
```json
{"main_picks":[{"priority":"⭐⭐","match":"利物浦 vs 阿森纳","pick":"主胜","odds":"1.72","confidence":"7/10","reason":"主场强势"}],
 "high_odds":[...], "parlays":[...],
 "lotto14":[{"no":"1","league":"欧冠","home":"阿森纳","away":"马竞","win":"1.64","draw":"4.03","lose":"5.60","pick":"3","dan":"是"}],
 "ren9":{"desc":"5胆4双=16注32元","picks":[...]}}
```

## 写入Excel命令

```bash
# 批量添加比赛到历史数据库
python3 ~/.hermes/scripts/write_excel.py add_matches '<json数组>'

# 添加复盘
python3 ~/.hermes/scripts/write_excel.py add_review '<json对象>'

# 读取今日推荐
python3 ~/.hermes/scripts/write_excel.py read_today

# 查看统计
python3 ~/.hermes/scripts/write_excel.py stats
```

## 写入竞彩分析库命令

```bash
# 写每日预测笔记
python3 ~/.hermes/scripts/obsidian_sync.py prediction 2026-05-05 '<json数据>'

# 写每日复盘笔记
python3 ~/.hermes/scripts/obsidian_sync.py review 2026-05-05 '<json数据>'

# 更新模型参数
python3 ~/.hermes/scripts/obsidian_sync.py model '<json数据>'

# 更新联赛统计
python3 ~/.hermes/scripts/obsidian_sync.py league '<json数据>'
```

## 竞彩分析库目录结构

```
桌面/竞彩分析库/
├── 索引.md               ← 库入口，自动更新
├── 每日预测/
│   ├── 模板.md           ← 格式模板
│   └── 2026-05-05.md     ← 每日预测笔记（自动生成）
├── 每日复盘/
│   ├── 模板.md
│   └── 2026-05-05.md     ← 每日复盘笔记（自动生成）
├── 模型参数/
│   ├── 当前参数.md        ← 模型权重、仓位参考、铁律因子
│   ├── 修正日志.md        ← 每错必修记录
│   └── 深盘预警库.md      ← 深盘历史表现跟踪
└── 联赛统计/
    └── 联赛基准率.md      ← 各联赛胜平负概率
```

## 每日自抓取流程（给 Cron Job 用的提示）

```
自动抓取500.com今日竞彩数据：
1. 打开浏览器访问 odds.500.com
2. 点击"竞彩足球"提取今日所有竞彩场次的赔率/盘口数据
3. 点击"足彩胜负"提取14场数据
4. 用 JavaScript 提取完整表格数据到结构化对象
5. 调用 daily_sync.py 一站式写入（总表Excel + 竞彩分析库）：
   /usr/bin/python3 ~/.hermes/scripts/daily_sync.py all 今日日期 '<比赛数据JSON>' '<分析数据JSON>'
6. 分析数据，按三层决策法给出推荐
7. 生成分析报告，包含主力推荐、14场分析、任9方案
```

## 注意事项

- 500.com 有反爬限制（Tencent Cloud EdgeOne），每分钟请求不超过10次
- 每次日期查询后至少等待5秒
- 自抓取定时任务使用 Hermes cronjob 系统，不需要额外安装 Playwright
- 使用 `browser_console` 执行 JavaScript 提取表格数据
- 竞彩分析库文件夹位于 Windows 桌面，使用 `/mnt/c/Users/Administrator/Desktop/竞彩分析库/` 路径
