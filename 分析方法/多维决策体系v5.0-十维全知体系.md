## 二、每个维度的深层资源

### 维度① 概率维度

**现有：** Dixon-Coles / Skellam / 有序Probit / BMA融合

**新发现的深层资源：**

| 资源 | 类型 | 价值 |
|:-----|:-----|:------|
| **arXiv:2507.10626** ⭐⭐⭐⭐⭐ | 论文 | Player-Team Heterogeneous Interaction Graph Transformer - **图神经网络预测足球赛果**，把球员和球队作为异构图节点，注意力机制捕捉相互影响 |
| **arXiv:2502.02785 (OpenSTARLab)** ⭐⭐⭐⭐ | 论文+代码 | 开源的时空足球分析平台，GitHub: open-starlab |
| **Gracenote ELO复现** ⭐⭐⭐⭐ | GitHub | github.com/torvaney/gracenote-elo - 商业模型的ELO复现版 |
| **StatsBomb xG方法论** ⭐⭐⭐⭐⭐ | 论文+数据 | 非参数贝叶斯xG模型，只用射门位置+身体部位+补射，不用防守变量（防过拟合） |

### 维度② 时空维度

**现有：** 时间衰减加权、日历效应(7个因子)

**新发现的深层资源：**

| 资源 | 价值 |
|:-----|:------|
| **OpenSTARLab时空分析** | 球员级别的时空追踪数据，分析跑位模式和攻防转换效率 |
| **Pinnacle开盘-关盘赔率曲线** | 开盘后30分钟存在大量错误定价 → 最佳下注窗口 |
| **Betfair交易所收盘价 = 真实概率** | 预测市场收盘价比任何模型更准 → 模型应拟合交易所价格而非庄家价格 |

### 维度③ 行为维度

**现有：** 行为金融反转策略(近期偏差/媒体热度/从众效应/后悔厌恶/锚定效应)

**新发现的深层资源：**

| 资源 | 价值 |
|:-----|:------|
| **澳客网必发指数** ⭐⭐⭐⭐⭐ | 国内唯一公开的庄家流动池数据 → 大额交易跟踪 → 反向做单信号 |
| **"Fade the Public"规则(Reddit)** | 当公众投注比例>80%时反向，胜率约61% |
| **球探网凯利离散度** | 各庄家凯利指数差异>0.15 → 庄家分歧大 → 常出冷门 |

### 维度④ 系统维度

**现有：** 联赛相关性、时间集中度、赔率集中度、冷门保护

**新发现的深层资源：**

| 资源 | 价值 |
|:-----|:------|
| **奥地利学派经济学视角** | 放弃英超等强有效市场，主攻知识密度低的边缘联赛（中甲/日J2/巴乙/缅甲） |
| **Bet365 Model V3泄露特征(40个)** | 传闻泄露的特征列表 → 我们的特征工程蓝图 |
| **Opta Top10特征** | 向前传球比例、高位拦截次数、禁区触球数、对手半场传球成功率 → 立即加入模型 |

### 维度⑤ 网络维度

**现有：** 无

**新发现的深层资源：**

| 资源 | 价值 |
|:-----|:------|
| **arXiv:2507.10626 异构图Transformer** | 球员-球队异构图 → 球员转会、伤病对球队网络结构的影响可以量化 |
| **传球网络分析** | 从传球数据构建球队传球网络 → 核心球员缺席=网络密度下降 |
| **球员交易网络** | 同一经纪人的球员→内部消息可能提前泄露 |

### 维度⑥ 事件维度

**现有：** 无

**新发现的深层资源：**

|:-----|:------|

### 维度⑦ 经济维度

### 维度⑩ 元认知维度

**知道自己不知道什么：**

| 盲区 | 原因 | 解决方案 |
|:-----|:-----|:---------|
| 球员实时体能状态 | 无法获取GPS数据 | 用上场时间+密集赛程代理 |
| 更衣室内部消息 | 不公开 | 关注赛前阵容发布会 |
| 裁判当晚状态 | 不公开 | 用裁判历史倾向代理 |
| 球队战术秘密 | 不公开 | 用传球网络代理 |
| 庄家内部模型 | 商业机密 | 用Bet365泄露的40个特征参考 |
| 早期历史数据(1993前) | 电子化程度低 | football-data.co.uk有1993起数据 |
