### ⑩ 蒙特卡洛路径模拟（Monte Carlo）

**理念：** 单场预测再准，多场联动时人脑也算不清。"某队不仅要赢还要另一场平局"这种复杂条件，让计算机模拟10万次最可靠。

**公式：**
```
P(条件成立) = 满足条件的模拟次数 / 总模拟次数(100,000)
每场模拟：从Dixon-Coles/Skellam比分分布中随机抽取
```

**落地应用：**
| 场景 | 怎么做 | 价值 |
|:-----|:-------|:-----|
| 串关EV校准 | 模拟串关10万次→真实中奖率 vs SP隐含率 | 发现价值偏差 |
| 14场胆材选择 | 模拟每场做胆材的可靠性 | 科学选胆非凭感觉 |
| 任9方案优化 | 模拟不同胆材组合的胜率 | 选最优方案 |
| 组合串关比较 | 模拟多种串关结构看哪个EV最高 | 优化投注策略 |

**与现有体系：** 复用Dixon-Coles的λ输出 → 作为第7个模型加入融合

### ⑪ 堆叠集成Stacking

**理念：** 现有BMA用固定权重平均，Stacking让机器学习最优权重——数据自己决定什么时候该听谁。

**与BMA对比：**
| | BMA(现有) | Stacking(新增) |
|:--|:----------|:---------------|
| 权重 | 固定值 | **数据驱动** |
| 训练 | 不入参 | 5折交叉验证 |
| 特征 | 仅模型概率 | 概率+场景特征 |
| 自适应 | 慢 | **每20场自动更新** |

**架构：**
```
Level-0: M1→M7 各输出主/平/客概率(共21维)
         + 场景特征(联赛/区间/完备度)
Level-1: 正则化逻辑回归 或 轻量XGBoost
训练：复盘数据 → 实际赛果(0/1)为标签
```

**门槛：** 20+复盘可初训，100+效果显著。前期BMA兜底

### ⑫ 行为金融反转策略

**理念：** 市场参与者不理性。识别系统性的认知偏差，反向操作。

**5种偏差：**

| 偏差 | 识别方法 | 操作信号 |
|:-----|:---------|:---------|
| 🔥近期偏差 | 近5场积分率 - 赛季积分率 | 差值>15%时反向 |
| 📰媒体热度 | 赔率波动幅度>10% | 波动大时等冷静后再投 |
| 🐑从众效应 | 官网支持率 vs 真实概率 | 支持率>70%时反向 |
| 😱后悔厌恶 | 冷门方SP偏高 | 冷门方EV>+10%可搏 |
| ⚓锚定效应 | 当前赔率距开盘偏离度 | 开盘方向与当前方向相悖时关注 |

**公式：**
```
行为信号 = 近期偏差×0.3 + 从众偏差×0.3 + 热度偏差×0.2 + 后悔偏差×0.1 + 锚定偏差×0.1

信号>0.5 → 🟢市场偏差明显，有利用空间
信号<-0.3 → 🔴市场基本理性
```

### ⑬ 日历效应

**理念：** 最简单的特征往往最有效。一个字段就能加。

**已验证的效应（25,168场数据）：**
| 因子 | 影响幅度 | 操作 |
|:-----|:--------:|:-----|
| 周中比赛 | 主胜率降8% | 周中仓位降档 |
| 国际比赛日后3天 | 强队主胜率降12% | 避强队或反向 |
| 赛季末最后5轮 | 无欲无求队爆冷率+15% | 战意权重×1.5 |
| 连续主场>2场 | 主场优势减半 | 主胜仓位降档 |
| 休息天数差>3天 | 每多1天优势+2% | 利用优势方 |
| 长假前后 | 注意力下降 | 减少博弈 |

**新增22维特征中的7个：** F16-F22全部来自日历

### ⑭ 投注心理学深层

**理念：** 除了行为偏差，更深层的心理机制也在影响市场。

**后悔厌恶（Regret Aversion）：**
```
现象：投资者更怕"做了后悔"而非"没做后悔"
在竞彩中的表现：热门方被过度追捧（怕买冷门错了被嘲笑）
利用：冷门方赔率常有+15~30%的溢价
规则：当冷门方EV>+10%且后悔厌恶信号>0.5 → 投注冷门
```

**确认偏差（Confirmation Bias）：**
```
现象：只看支持自己观点的信息
表现：买了某队后只关注有利于该队的新闻
利用：买定离手，复盘时再客观分析
```

**处置效应（Disposition Effect）：**
```
现象：过早卖出赢利的投资，死扛亏损的
在竞彩中：串关中了2场就急着对冲剩下的
利用：相信系统，不要凭感觉中途变卦
```

**落地：** 这些作为"心理纪律"加入SOP的第⑤层

### ⑮ 裁判倾向量化

**理念：** 裁判的执法风格（严格/宽松/偏主队）系统性地影响比赛。

**核心数据：**
```
每个裁判的画像：
  - 场均黄牌数（严格程度）
  - 主队胜率 vs 客队胜率（偏主程度）
  - VAR介入频率
  - 点球判罚倾向
```

**修正公式：**
```
修正后λ_home = λ_home × (1 + δ_referee_home_bias × 0.05)
修正后λ_away = λ_away × (1 + δ_referee_away_bias × 0.05)

其中δ_referee来自该裁判历史数据的贝叶斯估计
```

**落地：** 先从小联赛开始累积裁判数据（中甲/日职/韩K），50场以上裁判可入模

### ⑯ 球队网络图论

**理念：** 传球网络反映真实战术结构。中心性高的球队控场强，传球密度高的球队阵地战强。

**指标：**
| 指标 | 含义 | 对赛果影响 |
|:-----|:-----|:-----------|
| 传球网络密度 | 配合默契度 | 高密度→控场型→小比分 |
| 控球核心度 | 是否依赖某个核心球员 | 核心被锁→表现下降 |
| 传球熵 | 进攻方式多样性 | 高熵→难防守 |
| 攻防转换速度 | 由守转攻效率 | 快速转换→反击型→大比分 |

**与伤病量化结合：** 核心球员缺席 → 网络密度下降 → 模型自动调低预期

**数据来源：** 需要赛后传球数据（难度高），初期可以不依赖这个方向

### ⑰ 现代投资组合理论

**理念：** 凯利只优化单场。多场比赛同时下注时，收益之间有相关性，需要用马科维茨均值-方差优化。

**公式：**
```
最大化: 夏普比率 = w'τ / sqrt(w'Σw)
约束: Σwᵢ = 1, wᵢ ≥ 0

τ = 每场的期望收益率向量
Σ = 比赛间收益率协方差矩阵
w = 资金分配权重向量
```

**与凯利的关系：**
```
凯利 → 算每场的"最佳单注比例"
投资组合 → 算多场之间的"最佳分配比例"
两者互补：凯利告诉你怎么下注，投资组合告诉你怎么分配
```

**简化落地（不跑优化也能用）：**
```
相关性原则：
  - 同联赛同日比赛相关系数≈0.2-0.35 → 不超过2场
  - 同联赛不同日 → ≈0.1
  - 不同联赛 → ≈0.05
  - 不同日期 → ≈0.00

仓位分配 = 凯利比例 × (1 - 平均相关系数)
```

### ⑱ 统计套利/对冲

**理念：** 不同平台、不同时间点的赔率差价，可能存在无风险或低风险机会。

**三类套利：**

| 类型 | 原理 | 竞彩可行性 |
|:-----|:-----|:-----------|
| 跨平台套利 | A平台主胜2.10,B平台客胜2.10→锁定利润 | ❌竞彩独家定价,无跨平台 |
| 时间对冲 | 早盘买入,临场反向卖出 | ✅ 竞彩SP变化可做 |
| 串关对冲 | 串关中部分场次已出结果后对冲剩余 | ✅ 部分可用 |

**时间对冲规则：**
```
买入选主胜@1.56(早盘)
临场主胜@1.45(降了) → 如果剩余场次中有赔率上升的
→ 卖出(买客胜)或换方案 → 锁定部分利润
```

### ⑲ 赔率时间序列分析

**理念：** 赔率变化轨迹揭示市场信息流。异常波动（突然单边移动）预示有重大信息泄露。

**核心指标：**
| 指标 | 含义 | 信号 |
|:-----|:-----|:-----|
| 变化速率 | 单位时间赔率变化量 | 速率突然增大→有信息冲击 |
| 波动率 | 赔率上下波动幅度 | 波动率增大→市场不确定性高 |
| 方向一致性 | 各平台变化方向是否一致 | 不一致→分歧大,需谨慎 |
| 拐点检测(CUSUM) | 趋势反转点 | 拐点→市场情绪逆转 |

**落地规则：**
```
IF 赔率变化速率 > 正常值3倍 → ⚡突发消息，先暂停分析
IF 波动率 > 15% → 市场分歧大，仓位降1档
IF 方向一致性 < 60% → 各平台看法不一，谨慎
IF CUSUM检测到拐点 → 原方向可能反转
```

### ⑳ 天气量化

**理念：** 暴雨/高温/大风/高海拔直接影响比赛结果。

**量化公式：**
```
进球修正 = λ_base × exp(β₁×temp + β₂×rain + β₃×wind)

β系数（基于历史数据估计）：
  - 温度>30°C: λ×0.92（体能下降）
  - 大雨: λ×0.85（技术动作受影响）
  - 大风>20km/h: λ×0.90（精准传球下降）
  - 海拔>2500m: 主队λ×1.15, 客队λ×0.85
```

**数据来源：** OpenWeatherMap API（免费版够用），赛前3小时查询

**优先用于：** 解放者杯（高原客场）、北欧联赛（严寒）、露天球场

### ㉑ 注意力经济

**理念：** 人的注意力有限。媒体聚焦的球队、热门比赛会被过度关注，导致赔率偏差。

**量化指标：**
```
注意力因子 = 
  0.3 × 近期媒体提及率 
  + 0.3 × 话题度(Google Trends) 
  + 0.2 × 近期战绩偏差(同行为金融)
  + 0.2 × 投注量偏离度
```

**利用方式：**
```
IF 注意力因子 > 0.7（市场过度关注某队）
   AND 模型概率 < 市场概率（模型不看好）
   → 投注另一方向（反向操作）

IF 注意力因子 < 0.3（市场忽视某比赛）
   → 可能出现价值投注机会（庄家也容易忽视）
```

### ㉒ 实时贝叶斯更新

**理念：** 赛前概率为先验，赛中每发生一个事件（进球/红牌/射门）就更新后验。

**流程：**
```
赛前: P(最终比分) = Dixon-Coles先验分布
赛中t时刻: P(最终比分 | 事件) ∝ P(事件|θ) × P(θ)
        θ = 两队当前进攻强度(随时间衰减)
        
事件更新规则：
  - 每过1分钟无进球 → λ略微衰减
  - 进球 → λ重新校准(进球方强度降,对方升)
  - 红牌 → 该队λ×0.70
  - 点球 → 预期进球+0.76(但未罚进前)
```

**与现有体系：** 当前只做赛前分析，实时更新是下一个阶段
