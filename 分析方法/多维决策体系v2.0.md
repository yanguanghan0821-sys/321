## Step① — 14场截止时间：三源交叉验证

### 为什么不能错？
14场截止时间≠首场开球时间！每期官方公告的停售时间可能不同。

### 操作步骤
| 次序 | 动作 | 验证点 |
|:----:|:-----|:-------|
| 1 | sporttery.cn → 传统足彩 → 读取官方公告停售时间 | 记录精确到分钟 |
| 2 | 同一页面 → 详细表格底部 → 查各场最后截止时间 | 与公告一致？ |
| 3 | 500.com → 足彩胜负 → 查该奖期停售信息 | 交叉验证 |

**三源结果对照表：**
- ✅ 三源一致 → 采用
- 🟡 两源一致 → 取两源的，标记差异源
- 🔴 全不同 → 以sporttery.cn公告为准+警告

**推送时间 = 截止时间 - 1小时**
（注意竞彩和14场截止可能不同，需分别设任务）

## Step② — 数据采集：四角验证

| 角度 | 操作 | 防错目标 |
|:----:|:-----|:---------|
| ①完整扫描 | 列全部场次、检查编号连续 | 不漏场 |
| ②双重提取 | 同一页面提取两遍、对比 | 不抓错 |
| ③跨源交叉 | sporttery.cn vs 500.com 数量对比 | 不漏数 |
| ④波动标注 | 对比上次采集SP、标记变化幅度 | 不忽变 |

**数据检验清单：**
```
□ 竞彩场次数=预期数
□ 场次编号连续无跳号
□ 每场都有主SP/平SP/客SP
□ 每场都有让球数+让球SP
□ 双重提取一致
□ 已与500.com交叉验证
□ SP变化已标注
□ 球队名在映射表中
```

## Step③ — 9层分析体系

### 从5层到9层：决策维度全面升级

| 层级 | 名称 | 核心内容 | 新增 |
|:----:|:-----|:---------|:----:|
| ① | 市场解读 | 竞彩SP vs 欧赔三维对比+返还率+Sharp Money | — |
| ② | 信息采集 | 15维特征(ELO/泊松λ/伤病/时间衰减/战意/交锋等) | — |
| ③ | 精算模型 | M1-DixonColes / M2-Skellam / M3-有序Probit | — |
| ④ | BMA融合 | 贝叶斯5模型加权+不一致度检测 | — |
| ⑤ | 纪律执行 | 风险调整凯利+仓位+选场铁律 | — |
| ⑥ | **系统风险** | 联赛相关性/时间集中度/组合保护 | ✅新增 |
| ⑦ | **时序模式** | 休息天差/周中周末/密集赛程/FIFA病毒/凌晨场 | ✅新增 |
| ⑧ | **外部因素** | 天气/海拔/裁判/场地/德比/空场 | ✅新增 |
| ⑨ | **叙事情报** | 媒体热度/舆情方向/反指识别 | ✅新增 |

### 第⑥层：系统性风险（新）

| 风险维度 | 修正规则 |
|:---------|:---------|
| 同联赛同时开球≥3场 | 每+1场，该场仓位-10% |
| 同一时段开球≥5场 | 每+1场，仓位-5% |
| 集中低赔(SP<1.50) | 合计仓位≤30%预算 |
| 无高赔对冲项 | 总仓位-20% |
| 信息完备度<50%场>30% | 总仓位-30% |

### 第⑦层：时序模式（新）

| 因子 | 修正 |
|:-----|:-----|
| 周中比赛(周二-周四) | 主胜概率×0.92(爆冷率高15%) |
| 休息天数差≥3天 | 每多1天主胜+2% |
| 7天内3赛(密集) | 主胜×0.88 |
| 客队长途>2000km | 主胜+5% |
| 国际比赛日后 | 主胜×0.92(强队影响更大) |
| 凌晨场(03:00+) | 客队概率+3% |

### 第⑧层：外部因素（新）

| 因素 | 修正 |
|:-----|:-----|
| 大雨/大雪 | 总预期进球-0.5球 |
| 高海拔(>2500m) | 主胜×1.15 |
| 人工草皮 | 主胜+5% |
| 德比战 | 主胜×0.90(情绪因素) |
| 空场/低上座 | 主胜×0.85 |

### 第⑨层：叙事情报（新）

| 叙事类型 | 操作 |
|:---------|:-----|
| 媒体热炒(头条频率高) | 热门方向仓位降档 |
| "某某队N场不败" | 物极必反警惕 |
| 核心伤病报道量级>实际影响 | 可能借机压低赔率，反向看 |
| 舆论一边倒 | 换位思考，信任数据 |
| 舆论沉默 | 重点关注模型信号 |

## Step⑤ — 截止前推送

统一推送内容：
```
1. 查所有场次最新SP变化
2. 查首发阵容（按联赛对应时间）
3. 标记有变化的场次
4. 如有核心轮换→调整推荐
5. 无变化→确认原推荐不变
```

## Step⑥ — 复盘全链路：风险对照表

### 核心新增：风险预警对照表（连接预测与复盘）

```
| 场次 | 赛前风险标注 | 实际结果 | 风险应验？ | 学习点 |
|:----:|:------------:|:--------:|:---------:|:-------|
| XXX | ⚡风险信号 | ✅/❌ | ✅/❌ | 学到了什么 |
```

### 复盘10项全覆盖
```
✅ 1. 实际比分（精确可验证）
✅ 2. 预测 vs 实际 → ✅/❌/🟡
✅ 3. 总命中率+分玩法命中率
✅ 4. 偏差定位 → 9层中哪一层？
✅ 5. 错误根因（具体原因）
✅ 6. 风险对照 → 之前标注的风险是否应验
✅ 7. 最佳预测→为什么对？
✅ 8. 连续同类错误计数
✅ 9. 特征权重修正建议
✅ 10. 写入修正日志
```

### 偏差定位速查

| 代码 | 层 | 自查问题 |
|:----:|:---|:---------|
| ① | 市场解读 | 竞彩SP vs 欧赔偏离判断对了吗？ |
| ② | 信息采集 | 漏了伤病/阵容/战意？ |
| ③ | 模型预测 | DC/Skellam/Probit概率偏差？ |
| ④ | BMA融合 | 权重分配偏了吗？不一致度处理对了吗？ |
| ⑤ | 纪律执行 | 仓位合规吗？铁律遵守了吗？ |
| ⑥ | 系统风险 | 组合保护够吗？相关性考虑了吗？ |
| ⑦ | 时序模式 | 休息天差/赛程密度考虑了吗？ |
| ⑧ | 外部因素 | 天气/海拔/裁判这些硬条件漏了吗？ |
