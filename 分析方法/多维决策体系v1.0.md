## 二、第①层：市场解读（Market Reading）

### 1.1 竞彩SP vs 欧赔三维对比法（已验证）

| 维度 | 数据源 | 说明 |
|:-----|:-------|:-----|
| 竞彩固定奖金(SP) | sporttery.cn | 体彩中心定价，投注实际赔率 |
| Bet365欧赔 | 500.com | 主流庄家定价参考 |
| 百家欧赔均值 | 500.com | 市场共识赔率 |

**对比信号矩阵：**

| 竞彩SP vs 百家均值 | Bet365 vs 百家均值 | 信号 | 操作 |
|:------------------:|:------------------:|:----|:----|
| 竞彩 < 百家(小5%+) | B365 < 百家 | 🟢三家一致看好 | 正常跟 |
| 竞彩 < 百家(小5%+) | B365 > 百家 | ⚡竞彩压赔/欧赔分歧 | 重点核查，可能是诱导 |
| 竞彩 > 百家(大5%+) | B365 < 百家 | 🟡竞彩不看好/欧赔看好 | 价值投注机会 |
| 竞彩 < 百家(小10%+) | B365 = 百家(稳) | 🔴竞彩大幅压低 | 强烈诱导信号，反向防冷 |

**差异量化公式：**
```
偏离度 = (竞彩SP - 百家均值) / 百家均值 × 100%

|偏离度| > 5%  → 🟡 轻微异常，关注
|偏离度| > 10% → 🔴 明显异常，重点分析
|偏离度| > 20% → ⚡ 强烈诱导信号
```

### 1.2 庄家资金流解读（Sharp Money Tracking）

**核心原理（从Pinnacle专业交易理论）：**
- 庄家的盘口调整反映的是**大宗资金流向**而非散户
- 庄家不预测赛果，他们**预测投注者的投注方向**

**跟踪规则：**

| 盘口/赔率变化 | 资金流解读 | 操作建议 |
|:--------------|:-----------|:---------|
| 主胜赔率↓ + 升盘 | 庄家真实看好，Sharp Money看多主队 | 🟢 可跟 |
| 主胜赔率↓ + 水位↑ | 诱导主胜（吸引投注主队带高水） | 🟡 谨慎 |
| 主胜赔率↑ + 降盘 | 庄家/Sharp Money看空主队 | 🔴 不跟主胜 |
| 主胜赔率↑ + 水位↓ | 压低赔付做防御，真实看空 | 🔴 强烈不跟 |
| 盘口不变但赔率↓ | 控制赔付风险 | 🟢 正常看好 |
| 大幅变盘(+10%以上) | 突发消息(伤病/阵容) | 🟡 查原因再决定 |

> ⚠️ **重要：** 竞彩SP 通常一天更新3-5次（10:00/14:00/18:00/21:00），变化幅度远小于欧赔。竞彩的大幅变化(>10%)才是有效信号。

### 1.3 返还率分析（新增维度）

**公式：**
```
返还率 = 1 / (1/主胜SP + 1/平局SP + 1/客胜SP)
```

| 返还率区间 | 含义 | 操作 |
|:----------:|:-----|:-----|
| >94% | 竞彩对投注者有利，市场正常 | 🟢 正常分析 |
| 91-94% | 市场正常范围 | 🟢 正常 |
| 88-91% | 偏低，可能有冷门 | 🟡 谨慎 |
| <88% | 极低返还率，看好方向有陷阱 | 🔴 高度警惕 |

## 四、第③层：5种高级概率模型

### 4.1 M1：Dixon-Coles精校版

**核心：增加低比分修正因子τ(x,y;ρ)**

```
τ(x,y;ρ) 修正函数：
  0-0: τ = 1 - ρ × λ1 × λ2
  0-1: τ = 1 + ρ × λ1
  1-0: τ = 1 + ρ × λ2
  1-1: τ = 1 + ρ
  其他: τ = 1

ρ ≈ 0.12(五大) / 0.08(中等) / 0.05(低级别)
```

### 4.2 M2：Skellam进球差模型

**直接预测进球差d，用于亚盘分析：**

```
P(d=D) = e^{-(λ1+λ2)} × (λ1/λ2)^{D/2} × I_{|D|}(2√(λ1×λ2))
主胜 = P(D>0), 平局 = P(D=0), 客胜 = P(D<0)
让球-1主胜 = P(D>1)
```

### 4.3 M3：加权有序Probit

**用特征权重打分，不同视角的概率估算：**

```
综合得分 = Σ(特征值 × 特征权重)
主胜概率 = Φ(得分 - τ1)
客胜概率 = Φ(τ2 - 得分)
平局概率 = 1 - 主胜 - 客胜
```

### 4.4 M4：贝叶斯模型平均(BMA)

**融合层：**

```
P(最终) = w1×P(M1) + w2×P(M2) + w3×P(M3) + w0×P(市场)

初始权重：w1=0.30, w2=0.22, w3=0.15, w4=0.18(市场), w5=0.15(联赛基准)
动态调整：连续正确+0.01, 连续错误-0.01, 边界[0.05, 0.35]
```

### 4.5 不一致度检测

```
不一致度 = max(P模型) - min(P模型)
<0.15 🟢 一致 → 正常仓位
0.15-0.30 🟡 中度分歧 → 仓位降1档
>0.30 🔴 极度分歧 → 不投(或替代方案)
```

## 五、第⑤层：纪律与执行

### 5.1 风险调整凯利

```
CR.A = 调整置信度 = 1 - (1-历史准确率) × (1-信息完备度)
风险调整凯利 = 标准凯利 × CR.A × 0.25
最终仓位 = 风险调整凯利 × 当日预算
```

| 期望值(EV) | 信息完备度高(≥0.7) | 信息完备度低(<0.5) |
|:----------:|:------------------:|:------------------:|
| +20%+ | 5-7% | 2-3% |
| +10-20% | 3-4% | 1-2% |
| +5-10% | 1-2% | 0%(不投) |
| <+5% | 0%(记录不投) | 0%(记录不投) |

### 5.2 破产风险公式

```
RoR = (1 - EV)^{破产前可损失次数}
原则：单日亏损≤总资的5%, 连亏3天停, 盈利30%提取一半
```

### 5.3 选场铁律

| 条件 | 操作 |
|:-----|:-----|
| EV < +5% | 不投 |
| 不一致度 > 0.30 | 不投 |
| 深盘(让球≥2) | 仓位1/3 |
| 联赛首轮 | 仓位1/2 |
| 信息完备度 < 0.5 | 仓位降档 |
